Genera una barra de progreso en la salida estandar de error (stderr)

Note
La barra de progreso se actualiza cada vez que se llama a esta funcion Se utiliza stderr dado que esta salida no esta bufferizada. En caso de utilizar stdout, la barra de progreso no se actualizaria hasta que se llame a fflush(stdout), la cual puede producir un resultado inesperado y a su vez no es parte del estandar.
